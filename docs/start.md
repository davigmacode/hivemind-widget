# Getting Started

Before using this widget you need to register and subscribe to enterprise package on [Hivemind](//:app.hivemind.id)

## Installation

```
<!-- optional if using content widget with engagement basic or reactions -->
<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons" />

<!-- config api key -->
<script>window.hivemind = { apikey: 'your-api-key' }</script>

<!-- if you only need content widget -->
<script src="https://unpkg.com/hivemind-widget/dist/content.js"></script>

<!-- if you only need summary widget -->
<script src="https://unpkg.com/chart.js@2.7.0/dist/Chart.min.js"></script>
<script src="https://unpkg.com/chartjs-plugin-datalabels@0.1.0/dist/chartjs-plugin-datalabels.min.js"></script>
<script src="https://unpkg.com/hivemind-widget/dist/summary.js"></script>

<!-- if you only need trend widget -->
<script src="https://unpkg.com/hivemind-widget/dist/trend.js"></script>

<!-- if you need all widget -->
<script src="https://unpkg.com/chart.js@2.7.0/dist/Chart.min.js"></script>
<script src="https://unpkg.com/chartjs-plugin-datalabels@0.1.0/dist/chartjs-plugin-datalabels.min.js"></script>
<script src="https://unpkg.com/hivemind-widget/dist/bundle.js"></script>
```
